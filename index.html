Projects

  • OxfordHack - Deep Doc

    This is an application to assist doctors in diagnosing complicated diseases. I used TensorFlow to predict the likelihood of an individual patient to have each type of cardiac arrhythmia based on their medical data. It also has a Doctor Facebook Messenger chatbot that can ask and note some simple symptoms from a patient, as well as their severity and duration. We visualise this and present the results on a dashboard. You can read more about the project here.

  • Deep Learning for Chemistry (Molecule Energy) Prediction

    I implemented a neural network to predict the ground state energy of a molecule given the Coulomb matrix of that molecule. I used optimisation techniques such as Adam and Batch Normalisation - all of this was implemented from scratch in Numpy (Python). You can read more about the project here.
